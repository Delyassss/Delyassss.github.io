# 📚 StudyDeck - Flashcard Learning App

A powerful, offline-first flashcard app to help you study smarter and retain more.

**[🚀 Open StudyDeck](https://Delyassss.github.io)**

---

## ✨ Features

🎯 **Smart Flashcards**
- Create unlimited decks
- Edit & organize cards
- Beautiful flip animation
- Shuffle for randomized learning

📖 **Study Mode**
- Interactive flashcard interface
- Mark correct/wrong
- Track progress
- Get percentage scores

⏱️ **Pomodoro Timer**
- 25-minute work sessions
- 5-minute breaks
- Adjustable intervals
- Auto-switching between work/break

📊 **Statistics**
- Total cards & decks tracked
- Sessions completed
- Average scores
- Performance analytics

🌙 **Customization**
- Dark mode toggle
- Sound effects
- Fully offline capable
- Data stored locally (private)

💾 **Offline Support**
- Works without internet
- All data saved locally
- Service Worker caching
- Perfect for studying anywhere

2. Create a new deck
3. Add flashcards
4. Click "Study" to begin
5. Track your progress

## 💡 How to Use

### Create Decks
1. Click **➕ Create**
2. Enter deck name & description
3. Click **Create**

### Add Cards
1. Click a deck to edit
2. Enter question & answer
3. Click **Add Card**
4. Repeat!

### Study
1. Open a deck
2. Click **Study**
3. Flip card to see answer
4. Mark ✅ Correct or ❌ Wrong
5. See your score!

### Use Timer
1. Click **⏱️ Timer**
2. Adjust work/break times (optional)
3. Click **Start**
4. Auto-switches between work & break

### Check Stats
- Click **📊 Stats** to see progress
- Total cards, decks, sessions
- Average score percentage
