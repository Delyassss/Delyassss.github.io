## 🌐 Step 1: Launch Web Version (10 minutes)

### Pick Your Hosting:

#### **🎯 Netlify** (EASIEST)
1. Visit **https://netlify.com**
2. Sign up (free)
3. Drag & drop your 3 files
4. Done! Get your URL instantly
5. **URL:** `your-app.netlify.app`

#### **⚡ Vercel** (FASTEST)
1. Visit **https://vercel.com**
2. Sign up with GitHub
3. Import folder
4. Auto-deploys
5. **URL:** `your-app.vercel.app`

#### **💻 GitHub Pages** (FREE)
1. Create GitHub repo: `yourusername.github.io`
2. Upload 3 files
3. Enable in Settings → Pages
4. **URL:** `https://yourusername.github.io`

**Pick ONE and deploy today!**

## 📱 Step 2: Convert to Android (2-3 days)

### Quick Path (Online Converter - 5 minutes)
1. Go to **https://www.appstomaker.com**
2. Upload `studydeck-v5.html`
3. Download APK
4. Done!

### Professional Path (Google Play Store)

**Once you have APK:**

1. **Create Google Play Account**
   - https://play.google.com/console
   - Pay $25 (one-time)

2. **Prepare Store Listing**
   - App name, description, icon, screenshots
   - Privacy policy (free template available)

3. **Upload APK**
   - Click "Create new app"
   - Upload your APK
   - Fill in store listing
   - Add screenshots

4. **Submit for Review**
   - Takes 24-48 hours
   - Usually approved!

5. **🎉 LIVE on Play Store!**
