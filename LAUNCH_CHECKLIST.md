## 🚀 DEPLOYMENT STEPS (15 minutes)

### ⏱️ Step 1: Create GitHub Repo (2 min)

```
1. Go to https://github.com/new
2. Repository name: yourusername.github.io
   (Replace yourusername with YOUR GitHub username)
3. Description: "StudyDeck - Flashcard Learning App"
4. Select "Public" (important!)
5. Check "Add a README file"
6. Click "Create repository"
```

**Result:** Repo created ✅

---

### ⏱️ Step 2: Upload Files (3 min)

```
1. You're now on your repo page
2. Click "Add file" → "Upload files"
3. Drag & drop these 4 files:
   • studydeck-v5.html
   • manifest.json
   • sw.js
   • README.md
4. At bottom, click "Commit changes"
5. Wait 2 seconds for upload
```

**Result:** Files uploaded ✅

---

### ⏱️ Step 3: Enable GitHub Pages (2 min)

```
1. Click "Settings" (top of page)
2. Click "Pages" (left menu)
3. Under "Source":
   - Branch: main
   - Folder: / (root)
4. Click "Save"
5. Wait 1-2 minutes
```

**Result:** See message: "Your site is published at..."

**Your URL:** `https://Delyassss.github.io` ✅

## ⚠️ Common Issues & Fixes

### "Repository not found"
- Check username spelling (case-sensitive)
- Make sure repo name is `yourusername.github.io`
- Refresh GitHub page

### "App won't load (404 error)"
- Wait 2-3 minutes after enabling Pages
- Check files are in root folder
- Clear browser cache (Ctrl+Shift+Delete)

### "Styles look broken"
- Make sure `manifest.json` is in root
- Clear cache
- Try different browser

### "Offline not working"
- Service Worker needs time to activate
- Try after 24 hours
- Must be HTTPS (GitHub Pages has this ✅)
